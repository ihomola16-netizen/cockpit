# GitHub Pages - Cockpit v3

Toto je prva testovacia verzia v3.

## Subory na upload

Ak chces v3 nasadit na GitHub Pages, nahraj obsah priecinka `cockpit-v3` do rootu repozitara:

- `index.html`
- `styles.css`
- `app.js`
- `models.js`
- `market-regime.js`
- `coin-analysis-engine.js`
- `data-adapter.js`
- `scanner-engine.js`
- `trigger-engine.js`
- `paper-simulator.js`
- `journal-engine.js`
- `rules-engine.js`
- `storage.js`
- `.nojekyll`
- volitelne `README.md`

## Dolezite

Nenahravaj priecinok `cockpit-v3` ako podpriecinok, ak chces aby web bezal priamo na:

`https://ihomola16-netizen.github.io/cockpit/`

Nahraj obsah priecinka priamo do hlavnej zlozky repozitara.

## Aktualizacie

Pri dalsich upravach staci prepisat zmenene `.js`, `.css` alebo `index.html` subory.

Po uploade sprav v prehliadaci tvrdy refresh:

`Ctrl + F5`
