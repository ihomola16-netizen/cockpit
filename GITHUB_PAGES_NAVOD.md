# GitHub Pages upload - Cockpit v2

Toto je verzia pripravena na GitHub Pages. Na web nahraj obsah tohto priecinka, nie cely priecinok ako podpriecinok.

## Subory, ktore maju byt v hlavnej zlozke repozitara

- `index.html`
- `styles.css`
- `app.js`
- `.nojekyll`
- volitelne `README.md`

## Najjednoduchsi postup

1. Otvor GitHub repozitar `cockpit`.
2. V hlavnej zlozke zmaz stare subory stareho cockpitu.
3. Nahraj subory z priecinka `cockpit-v2`.
4. Pockaj 1-2 minuty.
5. Otvor adresu GitHub Pages a sprav tvrdy refresh.

## Co zmazat zo starej verzie

Ak su v repozitari tieto stare subory, mozes ich zmazat:

- `analysis.html`
- `analysis.css`
- `analysis.js`
- `landing.css`
- `manifest.webmanifest`
- `sw.js`
- stare assety alebo subory, ktore patria k v1

Ak si nie si isty, nechaj iba subory uvedene v prvej casti tohto navodu.

## Ako aktualizovat neskor

Pri dalsich upravach staci na GitHub znova nahrat prepisane:

- `index.html`
- `styles.css`
- `app.js`

GitHub Pages sa aktualizuje automaticky.
