# Cockpit v2

Novy samostatny prototyp. V1 ostava nedotknuty.

## Ciel v2

- Scanner ma hladat konkretne setup signaly, nie iba coiny.
- Kazdy signal uklada snapshot: cena, score, setup typ, entry, SL, TP, dovody.
- Paper trigger vie spustit test obchod bez Binance API a bez realnych penazi.
- Real trades su oddelene od paper trades.
- Coin analysis karta vysvetluje metriky cez male `i`.

## Ako spustit

Otvor `cockpit-v2/index.html` v prehliadaci.

## Poznamka

Toto je prva verzia v2 na ladenie architektury a workflow. Nie je to finalny trading nastroj.
